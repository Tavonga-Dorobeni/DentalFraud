# Codex -- FDCDF Project

You are working on the **Forensic Dental Claims Detection Framework (FDCDF)**.

## Required Reading

Before making any changes, read and follow these documents:

- `docs/AGENT.md` -- Architecture, conventions, and task patterns for the full system
- `docs/SPEC.md` -- Functional and non-functional requirements
- `docs/DESIGN.md` -- UI/UX direction, design system, component specifications

## Scope

**You are responsible for the backend only.**

- Work exclusively within `packages/backend/` and `packages/shared/` (for shared types).
- Do not modify anything in `packages/frontend/`.
- Follow the backend architecture, module structure, API conventions, and database patterns described in `docs/AGENT.md` Sections 3, 5, 6, 7, and 8.
