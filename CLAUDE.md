Before making any changes, read and follow these documents:

- `docs/AGENT.md` -- Architecture, conventions, and task patterns for the full system
- `docs/SPEC.md` -- Functional and non-functional requirements

- Follow the frontend architecture, Vue feature structure, and design system alignment described in `docs/AGENT.md` Sections 4, 7, and 8.
